# Financial Analysis Dashboard

## Project Overview

The Financial Analysis Dashboard is an interactive Power BI project designed to provide comprehensive insights into business financial performance. The dashboard helps analyze revenue, profit, sales trends, customer segments, and geographical performance through interactive visualizations and key performance indicators (KPIs).

This project demonstrates data analysis, data modeling, DAX calculations, and dashboard design skills to support data-driven business decisions.

## Objectives

* Monitor overall financial performance.
* Analyze revenue and profit trends.
* Identify high-performing and low-performing regions.
* Evaluate customer segment performance.
* Discover business growth opportunities through data insights.
* Support strategic decision-making using interactive dashboards.

## Tools & Technologies

* Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Microsoft Excel

## Key Performance Indicators (KPIs)

* Total Revenue
* Total Profit
* Profit Margin
* Total Orders
* Average Order Value
* Growth Metrics

## Dashboard Pages

### 1. Executive Summary

This page provides a high-level overview of business performance through key financial KPIs and summary visualizations.

Features:

* Revenue Overview
* Profit Overview
* Order Performance
* Financial KPI Cards
* Business Performance Summary

### 2. Detailed Analysis

This page focuses on in-depth financial analysis and trend evaluation.

Features:

* Revenue Trends
* Profit Trends
* Monthly Performance Analysis
* Category-Level Analysis
* Financial Comparisons

### 3. Geographic and Segment Analysis

This page analyzes business performance across regions and customer segments.

Features:

* Regional Performance Analysis
* Geographic Revenue Distribution
* Customer Segment Analysis
* Profit by Region
* Segment-wise Financial Insights

## Key Insights

* Revenue and profit trends help identify business growth patterns.
* Regional analysis highlights high-performing and low-performing locations.
* Customer segment analysis provides insights into revenue contribution across segments.
* Financial KPIs support strategic business planning and decision-making.
* Interactive filters enable deeper exploration of financial performance.
